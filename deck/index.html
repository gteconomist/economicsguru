Deck Chart Studio

Set the time range, theme, and variables for each chart, preview it live, then export the config that auto-updates your PowerPoint.

How this works

Each card is one chart from the shared registry (assets/js/chart-registry.js) — add a chart there and it shows up here automatically. Your choices are saved in this browser. Set an optional Slide # to map a chart to a slide in your deck, then Download deck_charts.yaml and run rebuild_deck.py --renderer browser to bake the current charts into the standalone deck.
